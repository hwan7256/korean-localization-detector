# SaaS 아이디어 제너레이터 — KLD 교훈 기반

## 배경

우리는 KLD(Korean Localization Detector)를 만들었다:
- 해외 마이크로 SaaS를 크롤링 → DeepSeek으로 한국 시장 적합도 분석 → 대시보드 제공
- 결과: 기술적으로는 작동하지만 사업적으로 실패

## 실패 원인 분석 (반면교사)

1. **AI 대체 가능성 높음**: "해외 SaaS 찾아서 분석해줘"는 Claude/ChatGPT 프롬프트 하나로 대체됨. 진입장벽 0.
2. **수동적 소비 모델**: 사용자가 직접 사이트에 올 이유가 없음. AI 에이전트가 스크래핑해가면 끝.
3. **광고 수익 불가**: 방문자가 없으니 광고 노출 0.
4. **바이럴 요소 없음**: "내 점수가 85점이야!" 같은 공유 동기 없음.
5. **수익 모델 불명확**: 누가 왜 돈을 내야 하는지 모름.

## 성공한 유사 모델의 공통점 (GTmetrix, PageSpeed Insights 등)

1. 사용자가 **자신의 URL을 직접 입력** → 필연적 방문 발생
2. **점수/등급**으로 직관적 피드백 → 공유 심리 유발
3. **개선 제안**을 주어 반복 방문 유도 ("고쳤더니 점수 올랐네?")
4. **무료 + 프리미엄** 구조가 명확: 기본 진단 무료, 상세 리포트 유료

## 우리가 가진 자산

- FastAPI + SQLite 백엔드 구축 경험
- Cloudflare + nginx + HTTPS 인프라
- DeepSeek/LLM API 연동 노하우
- 웹 크롤링/스크래핑 파이프라인
- 한국 시장 도메인 지식
- GitHub Pages 정적 배포 경험
- 모바일 반응형 대시보드 UI 제작 능력

## 지침

위 배경을 바탕으로, 아래 조건을 모두 만족하는 SaaS 아이디어 10개를 생성해라.

### 필수 조건
1. **사용자 능동 입력 필수**: URL, 파일, 텍스트 등 사용자가 뭔가를 입력해야 결과가 나오는 구조
2. **AI 대체 불가**: 단순 "분석해줘" 프롬프트로 재현할 수 없는 복잡성 필요 (실제 크롤링, 실시간 데이터, 멀티스텝 파이프라인)
3. **점수/시각화**: 3~5개 카테고리로 점수를 매기고 시각화하여 직관적 피드백 제공
4. **반복 방문 유도**: 개선 후 재측정, 주간 리포트, 트렌드 변화 추적 등
5. **명확한 수익 모델**: 무료 티어 + 유료 티어 경계가 명확할 것
6. **글로벌 타겟 가능**: 한국만이 아닌 글로벌 시장을 노릴 수 있을 것
7. **1인 개발 가능**: MVP를 혼자 2주 안에 만들 수 있는 규모
8. **B2C 또는 B2소상공인**: 개인이나 소규모 비즈니스가 타겟. 엔터프라이즈 영업 필요 없을 것

### 각 아이디어에 포함할 것
- **이름**: 제품명
- **한줄**: 15단어 내외
- **입력**: 사용자가 무엇을 입력하는가
- **분석**: 어떤 분석을 수행하는가 (LLM + 자체 로직 구분)
- **출력**: 4~5개 카테고리 점수 + 종합 점수 + 개선 제안
- **수익 모델**: 무료/유료 경계
- **경쟁사**: 유사 서비스 존재 여부
- **차별점**: 왜 AI 에이전트가 대체할 수 없는가
- **MVP 구현 난이도**: 상/중/하
- **예상 트래픽 소스**: SEO, 바이럴, 광고 등
- **리스트**: 왜 이 아이디어가 망할 수 있는가 (정직하게)

### 금지할 아이디어 유형
- ❌ 단순 리서치/리포트 생성기
- ❌ "AI가 알려주는 ~" 식의 LLM 래퍼
- ❌ 사용자 액션 없이 수동 소비하는 대시보드
- ❌ B2B 엔터프라이즈 영업이 필요한 제품
- ❌ 법적 규제 리스크가 큰 영역 (의료, 금융 인허가, 저작권 침해)

각 아이디어를 구체적인 작동 시나리오와 함께 설명해라. "사용자가 X를 입력하면 → Y를 분석해서 → Z를 보여준다" 식으로.
